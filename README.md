# PHASE 1 : DATA CLEANING & PREPROCESSING
Heart disease prediction system 
This project is designed to develop a Heart Disease Prediction System using machine learning techniques to determine the likelihood of a patient having heart 
disease based on medical data. The dataset includes several important attributes such as age, gender, chest pain type, blood pressure, cholesterol levels, 
and other health indicators. In the beginning, the data was carefully cleaned by handling missing values, removing duplicate entries, and converting categorical 
features into numerical form. Additional preprocessing steps like feature scaling were applied to improve model efficiency. We also performed basic exploratory 
data analysis to better understand the distribution and relationships within the dataset. The processed data was then divided into training and testing sets for 
model building. Two classification algorithms, K-Nearest Neighbors (KNN) and Logistic Regression, were implemented and tested. Parameters such as the value of K in
KNN were adjusted to optimize performance. The models were evaluated using accuracy scores, and a comparison was made to identify which algorithm performs better.
Overall, this project highlights how machine learning methods can be used effectively in the healthcare field for early prediction and analysis of heart disease
